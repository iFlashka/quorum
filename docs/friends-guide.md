# Гайд для друзей — как подключиться к Quorum

## Установка

1. Скачай последний `.msi` или `.exe` отсюда: [GitHub Releases](https://github.com/USER/quorum/releases/latest).
2. Запусти. Windows может выдать «Windows protected your PC» — это потому что у меня нет code-signing сертификата (купить стоит ~$200/год для пет-проекта). Жми **More info** → **Run anyway**.
3. Установщик распакует `Quorum.exe` в `Program Files` и сделает ярлык в Пуске.

## Первый запуск

При первом старте откроется окно онбординга:

1. **Адрес сервера**: спроси у владельца сервера. Что-то вроде `https://quorum.example.com`.
2. **Invite-код**: тоже спроси. Логин/пароль ты придумываешь сам при регистрации.
3. После регистрации сразу попадаешь в гилду, в которую тебя позвали.

## Что есть

- **Текстовые каналы** с упоминаниями (`@username`), реакциями, файлами до 25 МБ.
- **Голос 1-на-1** — наведись на участника справа в Members, кликни Phone.
- **Голосовые каналы** (Lounge и т.п.) — клик по voice-каналу присоединяет; повторный клик выходит.
- **Видео и screenshare** — кнопки в плашке активного звонка / голосового канала.
- **Push-to-talk** — Settings → «Голос и видео» → Push-to-talk + хоткей (по дефолту `Shift+Space`).

## Поведение окна

- Закрытие на красный крестик **сворачивает в трей**, а не выходит.
- Полный выход — из tray-меню (правый клик на иконке) → «Выйти».
- При @упоминании в чате прилетает native Windows toast (можно отключить в Settings).
- Бейдж непрочитанных: красная точка на tray-иконке + `Quorum • N` в заголовке taskbar.

## Автозапуск и обновления

- В Settings → «Уведомления» можно поставить «Запускать с системой» — тогда Quorum стартует свёрнутым в трей при логине.
- Обновления приходят автоматически. При появлении новой версии в правом нижнем углу всплывёт предложение «Установить и перезапустить».

## Если что-то не работает

- Камера/микрофон не пускают: Windows Settings → Privacy & security → Camera/Microphone — разреши Quorum.
- Голос не идёт совсем: проверь что у тебя не закрыты UDP-порты (бывает на корпоративных сетях). Попроси админа сервера проверить логи.
- Нечем заменить пароль если забыл: пока никак, попроси админа удалить твой аккаунт и заново зарегистрируйся.

## Багрепорты

В чат/личку владельцу сервера. Открыть DevTools в окне: `Ctrl+Shift+I` (включено в dev-сборках), скопировать вывод Console.
